# TopBar Setup

- [Zebar](https://github.com/glzr-io/zebar)

To configure Zebar:

1. Copy [vanilla-clear](./vanilla-clear) widget to Zebar directory

```
cp ./vanilla-clear/ ~/.glzr/zebar/
```

2. Then disable other widgets in Zebar system tray.
